#	Physarum polycephalum

![](https://i.imgur.com/QNRQ1qJ.png)

**Usage**: 
```
git clone https://github.com/s3dman/slimesimulation.git
cd slimesimulation
mkdir build && cd build/
cmake ..
cmake --build .
./slimesimulation
```
**NOTE:** Make sure you have installed raylib(https://github.com/raysan5/raylib)
